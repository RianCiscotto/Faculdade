1 - Dados são fatos brutos, valores sem contexto que, quando organizados e processados geram informação. Podem ser estruturados, como tabelas ou não, como áudios e imagens.
2 - Informações são fatos úteis que podem ser exttraídos de dados, agora tendo contexto e significado
3 - A diferença entre eles é que Dados são elementos brutos que não possuem significado próprio, já informação são dados que já foram organizados para ter significado.
4 - Dados são importantes na tomada de decisão pois eles auxiliam o usuário a obter mais informações, assim, podendo tomar uma decisão com mais estudo e fatos.
5 - Antes do banco de dados os dados eram armazenados de forma física ou digital, porém sem relacionamento entre eles, causando problemas e redundância.
6 - Banco de dados são usados para armazenar e organizar dados, de modo que seja mais facil e rápido seu gerenciamento e acesso
7 - - Banco de dados relacional: Armazena os dados em tabelas estruturadas, com colunas e linhas, usando SQL para manipulação (ex.: MySQL, PostgreSQL).Banco de dados NoSQL: Usa estruturas mais flexíveis, como documentos, chave-valor, colunas ou grafos, sendo ideal para grandes volumes de dados não estruturados (ex.: MongoDB, Cassandra).
8 - Banco de dados relacionais: MySql e PostgreSql
9 - Banco de dados relacionais são importantes no quesito eficiência, integridade e segurança.
10 - O relacional possui Colunas(atributos), como 'Nome', 'Idade'. Linha(Tupla). E o valor de cada atributo.
